# Devloop Living Documentation

This directory contains living documentation for the devloop plugin - documents that are meant to evolve with the project and serve as both a reference and a guide for development.

## What is Living Documentation?

Living documentation is:
- **Always current**: Updated as the plugin evolves
- **Actionable**: Provides guidance, not just descriptions
- **Connected**: References related documents and code
- **Accessible**: Written for both new and experienced users
